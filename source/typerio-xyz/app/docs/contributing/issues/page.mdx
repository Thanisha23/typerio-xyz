# Reporting issues

To report a problem, select the repository of the affected project:
- [typerio](https://github.com/Truezeber/typerio)
- [typerio-react](https://github.com/Truezeber/typerio-react)
- [typerio.xyz](https://github.com/Truezeber/typerio-xyz)

Open a thread, describe your problem, idea or question in detail, and use appropriate labels. Remember to describe in as much detail as possible what you have in mind, supporting the post with as much technical information as possible.

When replying in already existing threads, remember to follow the Code of Conduct and avoid straying from the topic. 

Thanks for your contribution to Typerio!
